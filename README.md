# Portfolio - Arthur Reis

Site de portfólio com tema claro/escuro e navegação suave. Todo o conteúdo está em `index.html`, com seções de apresentação, projetos, estudos e contato.

## Como usar
1. Abra `index.html` com o live server.
2. Use o menu para rolar pelas seções.
3. Alterne o tema no topo; a escolha fica salva no navegador.
4. Envie uma mensagem pelo formulário: o botão abre o cliente de email já preenchido para `arthurfreisp@gmail.com`.

## Conteúdo
- Sobre: nome, interesses e breve resumo.
- Projetos: Organiza Festa, Roguelike ASCII e FarmaCO.
- Estudos: links para Developer Roadmaps, Python Docs, beecrowd e GitHub, além de referências OldSchool/experimentais.
- Contato: email direto e links para LinkedIn/GitHub.

## Imagens
As fotos estão em `assets/img/`: `Arthur.jpg`, `Projeto-Organiza-Festa.jpg`, `Projeto-FarmaCO.jpg` e `Projeto-roguelike.jpg`.

## Stack e recursos
- HTML semântico (`header`, `nav`, `main`, `section`, `article`, `footer`).
- Layout responsivo com Flexbox e Grid.
- JavaScript: alternância de tema com persistência, ano automático no rodapé e formulário que pré-preenche um email.
